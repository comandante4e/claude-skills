# Grill Me — версия для Codex CLI (GPT под капотом)

Это Codex-адаптация скилла `grill-me`; логика и процесс те же, что в [SKILL.md](SKILL.md) — здесь только дельта под Codex.

Скилл **чисто инструкционный**: это промпт, который заставляет агента допрашивать тебя по плану/дизайну по одному вопросу за раз, проходя дерево решений и разрешая зависимости между ними; на каждый вопрос агент даёт свой рекомендованный ответ; если на вопрос отвечает код — агент лезет в кодовую базу вместо вопроса. Ни субагентов, ни скриптов, ни данных, ни внешних инструментов он не использует. Под Codex **поведение переносится 1-в-1** — меняется только то, ГДЕ скилл живёт и как триггерится.

## Куда положить и как активировать

Codex автоматически читает `AGENTS.md` (корень проекта или `~/.codex/AGENTS.md`). Авто-триггера по `description` (как у Claude-скиллов) у Codex **нет** — текст инструкции нужно положить туда, откуда Codex его подхватит. Три способа:

1. **Правило в `AGENTS.md`** (по фразе-триггеру) — блок ниже в `AGENTS.md` проекта или в `~/.codex/AGENTS.md` глобально.
2. **Кастомный промпт для слэш-вызова** — положи текст в `~/.codex/prompts/grill-me.md`, тогда в чате доступен `/grill-me`.
3. **Вручную** — просто вставь текст инструкции в начало сессии.

Минимальный блок для `AGENTS.md`:

```md
## Grill me (допрос по плану/дизайну)
Когда пользователь говорит «grill me» / «прожарь меня» / «застресс-тести план» —
допрашивай его по каждому аспекту плана, пока не придёте к общему пониманию.
Иди по каждой ветке дерева решений, разрешая зависимости между решениями по очереди.
На каждый вопрос давай свой рекомендованный ответ. Вопросы — строго по одному за раз.
Если на вопрос можно ответить, изучив кодовую базу, — изучи код, а не спрашивай.
```

Содержимое `~/.codex/prompts/grill-me.md` — тот же текст инструкции из [SKILL.md](SKILL.md) (тело без YAML-фронтматтера).

## Что меняется против Claude-версии

Ничего в самом поведении. Единственная дельта — носитель и триггер (см. выше): `/grill-me` в Claude Code → правило в `AGENTS.md` или промпт-файл `~/.codex/prompts/grill-me.md` в Codex.

Пара заметок под GPT:
- **«Один вопрос за раз» — держать жёстко.** GPT под нагрузкой склонен вываливать список вопросов сразу; в Codex явно соблюдай: задал один — дождался ответа — следующий.
- **«Есть в коде — читай код, не спрашивай».** Чтение файлов Codex делает сам, инструментом не отличается от Claude — просто не ленись и не превращай это в вопрос пользователю.
