# Словарь русских ИИ-маркеров + чек-лист

Признаки, по которым русский текст читается как сгенерированный. Для каждого — что делать. Используй при технике 2 (анти-клише) и для финальной самопроверки.

> Правило поверх всех замен: **сначала пробуй удалить.** Если без штампа смысл не теряется — просто вырежи. Замена нужна, только когда слово несло реальную мысль.

## Слова и фразы-штампы

| Штамп | Что делать |
|---|---|
| в современном мире / в наше время / в эпоху цифровизации | убрать целиком, начать с сути |
| важно отметить / стоит отметить / следует подчеркнуть / нельзя не сказать | убрать, сразу факт |
| играет ключевую / важную / значимую роль | конкретный глагол: «определяет», «решает», «держит» |
| является неотъемлемой частью | «входит в», «без него не работает» |
| краеугольный камень / фундамент / основа основ | назвать прямо, что именно |
| открывает новые горизонты / возможности / перспективы | сказать, какие именно — конкретику |
| раскрыть (весь) потенциал | «позволяет сделать X» |
| выводит на новый уровень / поднимает на новую высоту | убрать или дать конкретный результат |
| меняет правила игры / переворачивает представление | убрать |
| не только… но и | разбить на два предложения |
| таким образом / следовательно / тем самым (в каждом абзаце) | проредить, оставить редко |
| в заключение / подводя итог / резюмируя | убрать, дать вывод напрямую |
| погрузиться / погружение в мир | «разобраться в», «заняться», «изучить» |
| является свидетельством / служит доказательством | «показывает», «доказывает» |
| служит маяком / путеводной звездой | убрать метафору |
| революционизировать / революционный | «изменить», «переделать», конкретику |
| бесшовный / бесшовно (seamless) | «без разрывов», «гладко» или убрать |
| комплексный / целостный подход | перечислить конкретные составляющие |
| широкий спектр / целый ряд / множество | назвать сколько и что именно |
| по-настоящему / поистине / действительно / по сути (усилители) | убрать |
| несомненно / безусловно / разумеется / очевидно | убрать (читатель решит сам) |
| когда речь заходит о / что касается | «у», «в», сказать прямо |
| в условиях постоянно меняющегося / динамично развивающегося | убрать |
| эффективное / оптимальное / идеальное решение | факт: что и насколько улучшает |

## Метафоры-кальки (образность из английского)

Гобелен, полотно (tapestry) · симфония / танец чего-либо · царство / мир / вселенная X (realm) · ландшафт (landscape) → «область», «рынок» · маяк (beacon) · кладезь · путешествие (journey, метафорическое).

**Действие:** убрать метафору, сказать буквально. «В постоянно меняющемся ландшафте маркетинга» → «В маркетинге, который меняется каждый год».

## Структурные маркеры

- **Правило тройки** — три прилагательных или три примера подряд («быстрый, надёжный и удобный»). → Оставь одно точное слово или сделай список разнородным по длине.
- **Шаблонная антитеза** — «дело не в X, а в Y», «это не просто X — это Y», «X — это не про A, это про B». → Разбей или убери; одна такая на текст максимум.
- **Абзацы-близнецы** — все одной длины и структуры. → Меняй длину (см. технику 1).
- **Жирный лид-ин в каждом пункте** списка. → Не везде; иногда обычный пункт.
- **Идеальная параллельность** конструкций. → Намеренно сломай симметрию.
- **Риторический зачин** — «Что если бы…?», «Задумывались ли вы…?», «Представьте:». → Начни с факта или сцены.
- **Заголовок над каждым абзацем.** → Группируй; не каждый абзац — раздел.

## Тональные маркеры

- **Безосновательный восторг:** «захватывающий», «потрясающий», «незабываемый», «уникальный». → Убрать или обосновать фактом.
- **Промо-язык:** «идеальное решение», «лучший выбор», «не упустите». → Заменить фактами и цифрами.
- **Хедж-вата:** «возможно, стоит рассмотреть», «может быть полезно», «в некоторых случаях». → Утверждай прямо или режь.
- **Псевдоглубина** без конкретики (абзац ни о чём, который звучит умно). → Добавь цифру/пример или вырежи целиком.
- **Вежливые мостики:** «Отличный вопрос!», «Давайте разберёмся», «Как мы видим». → Убрать.

## Пунктуация и оформление

- **Em-dash пачками** (—). Чаще ~1 на абзац — уже подозрительно. → Часть замени на запятые, точки, скобки.
- **Эмодзи** в деловом/экспертном тексте, если не просили. → Убрать.
- **Списки там, где просится проза.** → Разверни в связный текст (и наоборот, если стена текста — разбей).
- **Идеально ровные абзацы** одинакового размера. → Разнобой.

## Чек-лист самопроверки

Прогони после правки. Любое «нет» — вернись и поправь.

- [ ] В тексте нет слов и фраз из стоп-листа выше?
- [ ] Длина предложений скачет — есть и короткие в 3 слова, и длинные?
- [ ] Каждое утверждение конкретно (цифра, имя, пример, факт), а не оценка?
- [ ] Нет правила тройки и симметричных антитез подряд?
- [ ] Метафоры-кальки убраны, сказано буквально?
- [ ] Em-dash не чаще ~1 на абзац, эмодзи нет (если не просили)?
- [ ] Есть живой голос и позиция, а не нейтральная вата?
- [ ] Убраны усилители-пустышки и вежливые мостики?
- [ ] **Я не выдумал факты, цифры, цитаты, личный опыт?**
- [ ] **Смысл и данные оригинала сохранены?**
